# Password Generator in C &#128274;
&#128274; Random password generator built in C, with interactive terminal menu and input validation. (Windows and Linux version) &#128274; <br>
This generator has a interactive terminal menu that enable the user to choose the type of password they wants - only letters, only number, and a combination of both with special characters. Furthermore, it asks the user to define the length of the password, validating every step that the user takes, accepting only valid inputs at every step.
